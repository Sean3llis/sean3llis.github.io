Javascript

Javascript is popular in web development because it is implented across a wide array of browsers and allows for client-side interactions with the browser and user. Because of newer players in the space such as Node.js, the popularity of Javascript has also seen an upswing. It is also user-friendly for object-oriented-programming (OOP), which makes it a go-to choice for many web apps.
