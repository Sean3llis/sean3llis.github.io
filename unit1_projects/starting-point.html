Responsive Design and Fluid Layouts

Our Views on Responsive Design

By: Dustin Snyder + Sean Ellis

Responsive Design

Responsive design consists of 3 separate but inter-related principles which include:

  • Fluid Grid
  • Responsive Images
  • Media Queries

Fluid Layouts

Fluid layout is a design that is oriented in percentages instead hard pixels sizes which allows designs to scale with the browser.

Your Conclusion

This discipline was created in order to address the changing ecosystem of the web in which the number of devices and the format of those devices has changed exponentially. Browsers now need to adapt from small mobile browsers up to high definition wide screen devices.
